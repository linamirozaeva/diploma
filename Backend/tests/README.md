Backend для системы онлайн-бронирования билетов в кинотеатр. Проект разработан на Django + Django REST Framework.

## Технологии

- **Backend:** Django 4.2, Django REST Framework
- **База данных:** SQLite (разработка), PostgreSQL (продакшен)
- **Аутентификация:** JWT (djangorestframework-simplejwt)
- **QR-коды:** qrcode[pil]
- **Изображения:** Pillow
- **Тестирование:** unittest, requests, BeautifulSoup4

## Функциональность

### Для гостей (без авторизации)
- Просмотр списка фильмов
- Просмотр кинозалов
- Просмотр расписания сеансов
- Проверка доступности мест

### Для авторизованных пользователей
- Все возможности гостей
- Бронирование билетов
- Просмотр своих бронирований
- Отмена бронирований
- Получение QR-кода билета

### Для администраторов
- Все возможности пользователей
- CRUD операции с фильмами
- CRUD операции с кинозалами
- CRUD операции с сеансами
- Управление типами мест (VIP/обычные)
- Просмотр всех бронирований
- Статистика по бронированиям

## Установка и запуск

### Предварительные требования
- Python 3.10+
- Git
- Виртуальное окружение (рекомендуется)

### Пошаговая установка

```bash
# 1. Клонировать репозиторий
git clone <url-репозитория>
cd cinema-booking-backend

# 2. Создать и активировать виртуальное окружение
python -m venv venv

# для Windows (Git Bash)
source venv/Scripts/activate

# для Windows (CMD)
venv\Scripts\activate

# для Linux/Mac
source venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Применить миграции
python manage.py migrate

# 5. Создать суперпользователя
python manage.py createsuperuser
# Введите: admin, admin@example.com, admin123

# 6. (Опционально) Создать тестовые данные
python tests/debug_post.py

# 7. Запустить сервер
python manage.py runserver
